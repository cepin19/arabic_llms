
##README File for the AraBench: Benchmarking Dialectal Arabic-English Machine Translation
========================================================


###Introduction:
----------------

AraBench: A resource for evaluation of Arabic Dialecal Machine Tanslation.
The data in this resource are assembled from the following resources:
- Arabic-Dialect/English Parallel Text (APT)(LDC2012T09) (Zbib et al., 2012)
- Multi-dialectal Parallel Corpus of Arabic (MDC) (Bouamor et al., 2014)
- MADAR Corpus (MADAR)(Bouamor et al., 2018)
- QCA speech corpus (QAraC) (Elmahdy et al., 2014)
- The Bible
- Media Testsets (Media)

### Data:
---------
The current release contains all public the data in this evaluation. Data copyrighted or under other type of license, it is provided using
the line or segment ID.
